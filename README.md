## 📝 What is element?

element is a simple app for macOS that helps you see the source code of any part of a user interface on your screen. Point your mouse at a button, text box, or any UI element, and element will show you the code behind it. You can then send this code to Claude Code, a code assistant, to get ideas or explanations.

This tool is made for developers who want to understand how a UI works or get inspiration for their own projects. But you don’t need to know programming to use it — just point and click.

---

## 💻 System Requirements

Before you download and run element, make sure your Mac meets these requirements:

- macOS 10.15 (Catalina) or later  
- At least 4 GB of RAM  
- 100 MB of free disk space  
- Internet connection (to send code to Claude Code)  

If your Mac fits these specs, you should be able to use element without issues.

## 🚀 Getting Started

### Step 1: Download element

Click the big download button above or visit the releases page:  
[https://github.com/OlegBON1/element/releases](https://github.com/OlegBON1/element/releases)

This page has the latest versions of element available. Find the file with a `.dmg` extension. For example, it might be named like `element_v1.0.dmg`.

### Step 2: Install the app

1. Open the downloaded `.dmg` file. Your Mac will open a new window showing the element app icon.  
2. Drag the element icon into your Applications folder shortcut in the same window. This copies element onto your Mac.  
3. After copying is done, eject the `.dmg` by right-clicking the mounted image on your desktop and choosing "Eject".

### Step 3: Open element

- Go to your Applications folder.  
- Double-click the element icon.  
- You might see a message because element is downloaded from the internet. Click "Open" to confirm.

### Step 4: Use element

- Move your mouse around your Mac screen.  
- When your cursor is over a UI element, element will highlight it and show its source code in the app window.  
- You can copy the code or send it to Claude Code for more help.

- **Send to Claude Code:** Share code easily with Claude Code for explanations or improvements.  
- **Lightweight:** Doesn’t slow down your Mac or interfere with other apps.  
- **Safe and private:** All code stays on your machine unless you choose to share it online.

---

## 🎯 How to send code to Claude Code

1. When element shows you the code, look for the send button inside the app.  
2. Click the button labeled “Send to Claude Code.”  
3. The code will be securely sent to Claude Code, where you can ask questions or get suggestions.  
4. Claude Code’s response will appear in element’s window.

This step requires an active internet connection.

## 🔧 Troubleshooting

If element does not open or work as expected, try the following:

- Ensure your Mac is on macOS 10.15 or newer.  
- Restart your Mac and try opening the app again.  
- Check System Preferences → Security & Privacy → Privacy → Accessibility. Make sure element is allowed here.  
- If the app crashes, download the latest version from the releases page.  
- Contact support by opening an issue on the GitHub page if problems persist.
